NxDraw ライブラリ クラス階層図

概要

　CNxBMPImageLoader クラスは、CNxDIBImage オブジェクトの内容を Windows Bitmap File として保存する、CNxDIBImageSaver の派生クラスです。

　BMP へはアルファ値を保存する事ができませんので、CNxDIBImage オブジェクトが持つアルファ値は破棄されます。ただし、32bpp 形式の場合は、コンストラクタへ渡す引数によって、アルファ値を除いた 24bpp 形式、又は アルファ値を含む 32bpp 形式として保存する事ができます。

CNxBMPImageSaver::CNxBMPImageSaver()

概要

　CNxBMPImageSaver クラスのコンストラクタです。

書式

CNxBMPImageSaver(DWORD dwFlags = stripAlpha)

引数

DWORD dwFlags　フラグ

解説

　stripAlpha を指定すると、32bpp 形式の CNxDIBImage オブジェクトを、アルファ値を除いた 24bpp 形式として保存します(デフォルト)。
