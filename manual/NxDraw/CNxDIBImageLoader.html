概要

　CNxDIBImageLoader クラスは、画像を読み込んで CNxDIBImage オブジェクトを生成するクラスです。CNxDIBImageLoader 自体は抽象クラスである為、このクラスのオブジェクトを生成する事はできません。CNxDIBImageLoader の派生クラスには、CNxBMPImageLoader, CNxPNGImageLoader, CNxJPEGImageLoader, CNxMAGImageLoader, CNxSPIImageLoader があります。

　CNxDraw::LoadImage() 関数は、入力 CNxFile オブジェクトから、上記の CNxDIBImageLoader 派生クラスより対応クラスを自動的に検索して、作成された CNxDIBImage オブジェクトを返します。CNxDraw::LoadImage() 関数を使用すれば、個々の CNxDIBImageLoader 派生クラスを直接使用する必要はありません。しかし、必要とされる画像フォーマットが予め決まっている場合は、個々の CNxDIBImageLoader 派生クラスを直接使用した方が効率的です。

CNxDIBImageLoader::IsSupported()

概要

　サポートされている形式であるかを調べます。

書式

virtual BOOL CNxDIBImageLoader::IsSupported(LPCVOID lpvBuf, LONG lLength) const = 0

引数

LPCVOID lpvBuf　画像データの最初から 2048 byte を読み込んだバッファへのポインタ
LONG lLength　画像データの長さ

戻値

　展開可能(サポートされている)形式らば TRUE

解説

　lpvBuf 引数で示されるデータを調べて、展開可能な形式であれば、TRUE を返します。

CNxDIBImageLoader::CreateDIBImage()

概要

　画像を展開して、CNxDIBImage オブジェクトを返します。

書式

virtual CNxDIBImage* CNxDIBImageLoader::CreateDIBImage(CNxFile& nxfile) = 0

引数

CNxFile& nxfile　読み込み元 CNxFile オブジェクトへの参照

戻値

　CNxDIBImage オブジェクトへのポインタ。失敗ならば NULL。

解説

　返された CNxDIBImage オブジェクトが不要になったら、必ず delete 演算子で削除して下さい。
