概要

　CNxDIBImageLoader クラスは、CNxDIBImage オブジェクトを画像ファイルとして CNxFile オブジェクトへ書き出すクラスです。CNxDIBImageSaver 自体は抽象クラスである為、このクラスのオブジェクトを生成する事はできません。CNxDIBImageSaver の派生クラスには、CNxBMPImageSaver, CNxPNGImageSaver があります。

CNxDIBImageSaver::SaveDIBImage()

概要

　CNxDIBImage を CNxFile オブジェクトへ書き出します。

書式

virtual BOOL CNxDIBImageSaver::SaveDIBImage(CNxFile& nxfile,
const CNxDIBImage& srcDIBImage,
const RECT* lpRect = NULL) const = 0

引数

CNxFile& nxfile　保存先オープン済み CNxFile オブジェクトへの参照
const CNxDIBImage& srcDIBImage　保存される CNxDIBImage オブジェクトへの参照
const RECT* lpRect　保存される CNxDIBImage オブジェクトの矩形(NULL ならば全体)

戻値

　成功ならば TRUE

解説

　書き込み先ファイルのモードは CNxFile::modeWrite (新規作成ならば CNxFile::modeCreate も)が含まれていなければなりません。この関数は、書き込み終了時にファイルを閉じません。明示的にファイルを閉じるには、CNxFile メンバの Close() 関数を呼び出して下さい。
