概要

　CNxScreen クラスは CNxWindow の派生クラスです。CNxWindow クラスの対象がクライアントウィンドウであるのに対し、CNxScreen クラスはサイズ固定のフレームウィンドウが対象です。フルスクリーンへの切り換えもサポートします。

　CNxScreen クラスでは、ウィンドウを CNxScreen::SetScreenMode() 関数で指定されたサイズへ固定しようとします。::SetWindowPos() 関数等を使用して、ウィンドウのサイズを一時的に変更する事は可能ですが、画面モードを切り替えるまでの間のみ有効となります。

　CNxScreen クラスを使用するには、コンストラクタでオブジェクトを構築した後、CNxWindow クラスメンバの CNxWindow::Attach() 関数を呼び出します。デフォルトでは、640x480 ウィンドウモードが使用されます。以前のバージョンと異なり Attach() 関数を呼び出した時点ではウィンドウのサイズ等の変更は行いません。また、CNxScreen::SetScreenMode() 関数は、Attach() 関数より前には呼び出せません。ウィンドウのサイズや解像度の変更は、CNxScreen::SetScreenMode() 関数を呼び出した時に行われます。CNxScreen::SetScreenMode() 関数を呼び出さない場合、CNxScreen クラスの動作は CNxWindow クラスと全く同じです。

　指定の画面モードが使用可能であるかを調べるには CNxScreen::IsCanUseWindowMode() と、CNxScreen::IsCanUseFullScreenMode() を使用します。これらは、static 関数ですので、CNxScreen クラスを構築する前に呼び出す事ができます。

　CNxScreen クラスが動作するには 16bpp (high color) 以上の色数を表示可能な画面モードが必須です。フルスクリーンのみを使用する場合でも、CNxScreen クラスではフルスクリーンへの切り替え時に、現在のデスクトップの色数を使用する(色数の切り替えを行わない)為、デスクトップの色数は 16bpp 以上でなければなりません。

　画面モードは、CNxScreen::SetScreenMode() 関数を呼び出す事によっていつでも切り替え可能です。

　フルスクリーンの状態で、アプリケーションが非アクティブにされた時の動作は DirectDraw によるフルスクリーン化と異なります。CNxScreen クラスでは非アクティブにされた場合、CNxScreen::SetScreenMode() 関数を呼び出して、ウィンドウモードへ切り替えるだけです。DirectDraw の様にウィンドウの最小化は行いません。また、再びアクティブ状態になってもフルスクリーンへの切り替えは行いません。

CNxScreen::SetScreenMode()

概要

　画面モードの変更を行ないます。

書式

BOOL CNxScreen::SetScreenMode(UINT uWidth, UINT uHeight, BOOL bFullScreen)

BOOL CNxScreen::SetScreenMode(BOOL bFullScreen)

引数

UINT uWidth　画面の幅
UINT uHeight　画面の高さ
BOOL bFullScreen　フルスクリーンへ切り替えるなら TRUE

戻値

　成功ならば TRUE

解説

　要求された画面モードへ切り替えます。要求された画面モードと、現在の画面モードが同じならば何も行なわずに TRUE を返します。bFullScreen 引数を TRUE にするとフルスクリーン、FALSE ならばウィンドウモードへ切り換えます。要求された画面モードへ切り換えられなかった時は FALSE を返します。

　引数が一つだけの関数は、解像度は変更せず、フルスクリーンとウィンドウを切り換えるのに使用できます。

　以前のバージョンと異なり、CNxScreen::Attach() 関数より前に、呼び出すことはできません。実際に画面モードを切り替える前に、目的のモードが使用可能であるかを調べるには、CNxScreen::IsCanUseWindowMode() 又は CNxScreen::IsCanUseFullScreenMode() 関数を使用して下さい。

CNxScreen::IsCanUseWindowMode()

概要

　ウィンドウモードが使用可能であるかを返します。

書式

static BOOL CNxScreen::IsCanUseWindowMode()

引数

　なし

戻値

　現在の解像度でウィンドウモードが使用可能であれば TRUE を返します。

解説

　この関数から FALSE が返される理由は一つで、現在のデスクトップの色数が 16bpp 以下の場合のみです。

CNxScreen::EnableAutoHideMenuBar()

概要

　フルスクリーン時にメニューバーを自動的に隠す機能を、有効又は無効にします。

書式

BOOL CNxSprite::EnableAutoHideMenuBar(BOOL bAutoHide)

引数

BOOL bAutoHide　有効にするならば TRUE

戻値

　以前の状態

解説

　bAutoHide 引数へ TRUE を渡すと、メニューバーを自動的に隠す機能が有効になります。マウスカーソルを画面の最上部へ移動する(又はキーボードの Alt キーを押す)とメニューが出現します。その後マウスカーソルがメニューバーから遠ざかるか、メニューを決定するとメニューは消えます。
